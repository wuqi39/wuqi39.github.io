Springboot 入门

发表于 2024-12-07 更新于 2025-05-16 分类于 编程 本文字数： 639 阅读时长 ≈ 2 分钟
第一个Spring Boot程序 —— Hello world

一、概述

1.1 什么是SpringBoot？

SpringBoot是在Spring的基础上发展而来的全新开源框架，其目的是为了简化部署Spring应用程序的配置过程，简单、灵活、便捷

1.2 特点

  1. 代码简洁，不臃肿
    • 注解驱动编程
    • 自动创建各种工厂类
    • 依赖注入
  2. 配置简单
    • 自动部署：自带Tomcat服务器，项目启动时完成资源部署
    • 易于单元测试：自带JUnit单元测试框架，可直接测试组件方法
    • 集成流行第三方框架/软件

二、环境准备

Java语言的软件开发工具包（Java Development Kit 即 JDK）：至少需要 JDK 8 或更高版本。
构建工具：可选择 Maven 或 Gradle 作为构建工具。
IDE：常见的 IDE 包括 IntelliJ IDEA、Eclipse 等。

三、初始化项目步骤

3.1 打开 IntelliJ IDEA，创建新项目

启动 IntelliJ IDEA，选择 “Create New Project”。

3.2 进入项目配置界面

选择 “Spring Initializr” 选项，然后点击 “Next”。

3.3 配置项目基本信息

  • 服务器 URL：使用默认 URL。
  • 项目名称：自由命名（如 hello-world，避免中文）。
  • 项目位置：选择存放文件夹。
  • 语言：选择 Java。
  • 类型：选择 Maven。
  • Group（组）：输入如 com.example。
  • Artifact（工件）：输入如 helloworld。
  • 软件包名称：自动生成（如 com.example.helloworld）。
  • JDK：选择已有的 JDK 版本（如 JDK 17）。
  • 打包方式：选择 Jar。

3.4 选择相关依赖项

在依赖项选择界面，搜索并选择 Spring Web。确保选中的 Spring Boot 版本为 3.3 或最新版本。

3.5 点击创建按钮

点击 “Finish” 完成项目的创建。

四、编写 Hello World 程序

4.1 编写控制器代码

创建一个与 SpringbootHelloApplication 文件同级的 controller 类（.java 文件），命名为 HelloController，代码如下： 

package com.cqw.helloworld;  

import org.springframework.web.bind.annotation.GetMapping;  
import org.springframework.web.bind.annotation.RestController;  

@RestController  
public class HelloController {  

    @GetMapping("/hello")  
    public String hello() {  
        return "Hello, World!";  
    }  
}

4.2 编写主应用代码

在同一目录下，扩展 HelloWorldApplication 文件（若不存在则右键创建），代码如下： 

package com.cqw.helloworld;  

import org.springframework.boot.SpringApplication;  
import org.springframework.boot.autoconfigure.SpringBootApplication;  

@SpringBootApplication  
public class HelloWorldApplication {  

    public static void main(String[] args) {  
        SpringApplication.run(HelloWorldApplication.class, args);  
    }  
}

4.3 运行程序

  1. 启动应用程序
    找到 HelloWorldApplication 类，点击类上方的绿色运行按钮（或右键选择 Run 'HelloWorldApplication.main()'）。
  2. 访问 Hello World
    启动成功后，打开浏览器，输入 http://localhost:8080/hello，页面将显示 “Hello， World！”。

五、总结

通过以上步骤，即可成功创建一个 Spring Boot 程序！
可尝试在 HelloController 中添加更多 API 接口，进一步探索 Spring Boot 的强大功能。
